Changes in v3.00.a, introduced in 11.2

11.2 - Changes in VHDL/Verilog/Netlist sources (.vhd, .v, .ngc, .edn) 


New Features:
  • MDIO interface for PHY register access
  • Internal loopback support
  • PLB Burst support. Updated library plbv46_slave_single_v1_01_a to plbv46_slave_burst_v1_01_a.
Resolved Issues:
  • Resolved issue with zero data padding when the frame size is less than minimum
    • Versions that have this issue: v2.01.a,v2.00.a
  • Updated the core to drop the incoming packet with frame size less than the minimum packet length(Runt Frame)
    • Versions that have this issue: v2.01.a,v2.00.a
  • Fixed issue with IFG(Inter Frame Gap) not meeting for half duplex mode operation
    • Versions that have this issue: v2.01.a,v2.00.a
  • Fixed issue with retry of packets in case of collision. Core is updated to retry the packet 16 times if collision is detected for all transmited packets.
    • Versions that have this issue: v2.01.a,v2.00.a
Known Issues /Limitations:
  • None

11.2 - Changes in tool interface files(.mpd)


  • Added new parameter C_INCLUDE_MDIO
  • Added new parameter C_INCLUDE_INTERNAL_LOOPBACK
  • Added new port PHY_MDC
  • Added new port PHY_MDIO
  • Added attribute BUFFER_TYPE=IBUF on PHY_tx_clk and PHY_rx_clk

11.2 - Changes in documentation associated with core


  • Updated the DS for resolved issues and for new parameter addition
  • Added MDIO interface register description
